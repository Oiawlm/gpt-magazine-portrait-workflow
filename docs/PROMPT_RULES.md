# Prompt Rules

## 人物一致性

每个任务都必须包含人物核心身份约束：

- 人种、年龄段。
- 身高、体重、体型。
- 脸型、五官、发型。
- 气质。
- 不可改变项。

固定人物时，人物像比风格更重要。

## 风格提示词

每个风格提示词至少包含：

- 服装。
- 场景。
- 姿态。
- 构图。
- 光影。
- 色调。
- 排版和文字。
- 避免项。

## 文字语言

文字语言服从画面气质。

- 市井纪实、烟火生活、中文口号、新中式语境，优先中文主视觉。
- 高奢封面、国际时尚、男刊、镜面红绳、商业肖像，优先英文或人物拼音。
- 不要机械把所有中文都改成英文。

## 字母密度

背景超大字要考虑密度。

- 短拼音可能显空。
- 长英文或长拼音更容易形成高级封面压迫感。
- 非姓名词如果版式成功，可以保留为视觉样张，不必强行覆盖。

## 文字安全边距

提示词可加入：

```text
顶部刊名和背景大字必须完整显示在画面安全边距内，不得被裁切、遮挡或出血到画面外；所有封面文字保持清晰可读，避免超大标题压出画布。
```

## 常用负向提示词

```text
女性化，瘦弱，尖脸，人物不像，文字乱码，画面裁切，肢体变形，模糊，多余人像，杂乱背景，廉价感
```

## 省额度

- 不为测试而生图。
- 不重复生成已经验证的方向。
- 用户拖图并触发工作流视为授权；正常新人物运行不再二次确认。
- 先让 Doubao-Seed-2.0-Pro 生成提示词队列，校验通过后由 Codex 自动执行第一轮默认任务。
- 输出会覆盖旧图、前置能力缺失或队列无法修复时才暂停。
- 先让 Doubao-Seed-2.0-Pro 生成提示词队列，必要时让 DeepSeek V4 Pro 做纯文本整理，最后由 Codex 生图能力执行。
