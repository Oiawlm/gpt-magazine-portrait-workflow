# GPT Image Guide

## 定位

本文件记录本项目的最终生图执行规则。仓库继续以 GPT Image 效果作为视觉质量基准，但标准执行路线是由 Codex 生图能力完成多视图参考图和最终杂志写真生成。

禁止将控制浏览器、操作 ChatGPT 网页版或 GPT 桌面端作为本项目工作流、fallback、增强能力或未来规划。

## 自动执行前检查

用户拖图并触发 `gpt-magazine-portrait` 工作流，视为授权执行当前 MVP 默认流程。执行前只做机器检查：

- 任务数量。
- 任务 ID。
- 人物参考图路径。
- 风格参考图路径。
- 输出文件名。
- 是否覆盖旧图。
- 本次由 Codex 生图能力执行。

如果输出会覆盖旧图，或关键路径/前置能力缺失，暂停并说明原因。正常新人物运行不再二次询问是否开始。

多视图参考图必须由 Codex 生图能力生成。原始照片横向拼版、截图拼版或手工拼接图不算成功结果；Codex 生图工具服务器错误、超时或不可用时，停在多视图阶段并提示稍后重试，不得创建拼版 fallback，不得继续进入 Doubao 队列或最终写真。

## 执行中

- 按校验通过的任务队列生成。
- 不做额外测试图。
- 第一轮默认 4 张；如状态不稳定，可由 Codex 内部分批处理。
- 如果一张图画面很好但文字不对，优先定向修正，不要无限重跑。

## 保存后

保存到对应人物目录，并检查：

- 文件存在。
- 文件大小正常。
- 图片能打开。
- 人物 Markdown 已更新。
- 任务队列状态已更新。
- 运行记录已更新。

## 常见问题

### 文字漂移

模型可能把小字放大，或把 `MODERN ISSUE` 之类的边角字变成主视觉。解决方式：

- 减少小字要求。
- 明确“最大背景字只能是某个词”。
- 如果小字无法准确呈现，宁可省略。

### 中文和英文选择

不要机械认为英文更高级。

- 市井烟火、中文口号、新中式语境：优先中文。
- 高奢封面、男刊、商业肖像、红绳镜面：优先英文或拼音。

### 背景大字密度

背景大字需要字母数量支撑版式。短词可能显空，长词更容易形成密集时尚感。
