# GPT Image Guide

## 定位

GPT Image / ChatGPT Plus 是本工作流默认最终出图层。提示词整理可以交给 Doubao 或 DeepSeek，但最终图片质量以 GPT Image 为主要目标。

## 执行前

必须确认：

- 任务数量。
- 任务 ID。
- 人物参考图路径。
- 风格参考图路径。
- 输出文件名。
- 是否覆盖旧图。
- 是否允许浏览器或桌面端 fallback。

## 执行中

- 每次只生成用户确认的任务。
- 不做额外测试图。
- 如果一张图画面很好但文字不对，优先在同一对话里做局部修正。
- 不要为了追求绝对文字准确无限重跑。

## 保存后

保存到对应人物目录，并检查：

- 文件存在。
- 文件大小正常。
- 图片能打开。
- 人物 Markdown 已更新。
- 任务队列状态已更新。
- 运行记录已更新。

## 常见问题

### 文字漂移

模型可能把小字放大，或把 `MODERN ISSUE` 之类的边角字变成主视觉。解决方式：

- 减少小字要求。
- 明确“最大背景字只能是某个词”。
- 如果小字无法准确呈现，宁可省略。

### 中文和英文选择

不要机械认为英文更高级。

- 市井烟火、中文口号、新中式语境：优先中文。
- 高奢封面、男刊、商业肖像、红绳镜面：优先英文或拼音。

### 背景大字密度

背景大字需要字母数量支撑版式。短词可能显空，长词更容易形成密集时尚感。
